## 📌 Descripción General

**SGO (Sistema de Gestión de OVAs)** es una solución tecnológica disruptiva diseñada para trascender el contenido educativo estático. Mediante la integración de **IA Generativa** y una arquitectura de **Generación Aumentada por Recuperación (RAG)**, el sistema transforma los Objetos Virtuales de Aprendizaje en ecosistemas adaptativos.

A diferencia de un repositorio tradicional, SGO personaliza la experiencia del aprendiz en tiempo real, automatizando la carga curricular y la evaluación de evidencias, permitiendo que el instructor actúe como un mentor estratégico en el proceso de formación técnica.

---

## 🎯 Objetivos del Proyecto

### Objetivo General
Implementar un ecosistema de aprendizaje inteligente basado en modelos de lenguaje de gran escala (LLM), que optimice la apropiación de competencias técnicas en el programa ADSO mediante la generación dinámica de contenido pedagógico y rutas de aprendizaje personalizadas.

### Objetivos Específicos
* 🏗️ **Arquitectura Robusta:** Diseñar e implementar una infraestructura escalable que integre motores de IA con bases de datos relacionales y almacenamiento en la nube.
* 🧠 **Inteligencia Adaptativa:** Desarrollar un motor RAG capaz de contextualizar las respuestas del LLM con el diseño curricular oficial del SENA, garantizando precisión técnica.
* 🚀 **Automatización DevOps:** Establecer un pipeline de CI/CD para asegurar despliegues continuos, versionamiento semántico y alta disponibilidad del sistema.
* 📊 **Validación de Impacto:** Evaluar el desempeño del sistema mediante pruebas de usuario (UAT) y métricas de precisión en la generación de contenido pedagógico.
